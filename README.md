# Vem pra Arena

[![Status](https://img.shields.io/badge/status-em%20desenvolvimento-brightgreen)](https://github.com/seu-usuario/vem-pra-arena)
[![Cesar School](https://img.shields.io/badge/Cesar%20School-POO-orange)](https://www.cesar.school)

> **Vem Pra Arena** é uma plataforma digital integrada para a Arena Pernambuco, projetada para transformar a experiência do cidadão e otimizar a gestão administrativa do estádio. O ecossistema une a conveniência da compra de ingressos com um sistema de engajamento gamificado e inteligência de dados para os gestores.

## Funcionalidades

- **Exploração e Recomendação** – Feed personalizado de eventos com filtros por categoria, data e disponibilidade.
- **Ticketing Digital** – Compra rápida (PIX, cartão) e geração de QR Code dinâmico. Carteira digital "Meus Eventos".
- **Gamificação e Apostas** – Usuário ganha moedas fictícias e aposta em resultados esportivos.
- **Loja de Recompensas** – Troca de moedas por benefícios reais (ingressos, tours, estacionamento).
- **Dashboard Administrativo** – Painel exclusivo para gestores com estatísticas de ocupação, análise de público e cadastro de eventos.
